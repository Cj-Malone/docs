Points are awarded for each task that is completed as follows:

| Completion Status | Points Awarded |
| ----------------- | :------------: |
| Fixed             |       5        |
| Not an Issue      |       3        |
| Already Fixed     |       3        |
| Too Hard          |       1        |
| Skipped           |       0        |
